Sensores
/api/sensors

Para obter e enviar dados dos sensores para a plataforma, você poderá utilizar este endpoint e os métodos HTTP abaixo.

  • GET
  • POST

Ao realizar uma requisição com o método GET, você poderá obter a seguinte resposta:

  • 200

Se houver registros no banco de dados:

[
    {
        "model": "core.sensor",
        "pk": 1,
        "fields": {
            "rain_level": 40,
            "soil_moisture": 68,
            "humidity": 69.0,
            "temperature": 25.8,
            "dew_point": 19.68,
            "heat_index": 27.0,
            "air_quality": 1797.31,
            "atmospheric_pressure": 92835.0,
            "water_flow": 0.0,
            "amount_water": 928.3,
            "percentage_water": 92.83,
            "luminosity": 23.81,
            "ultraviolet_radiation": 1.08,
            "created_at": "2023-01-22T00:58:42.701",
            "updated_at": "2023-01-22T00:58:42.701"
        }
    }
]
Atributo Descrição Tipo
model Nome da classe no banco de dados. Texto
pk Identificador do registro. Inteiro
fields Campos da classe no banco de dados. Objeto
rain_level Nível de chuva em porcentagem. Inteiro
soil_moisture Nível de umidade do solo em porcentagem. Inteiro
humidity Nível de umidade do ar em porcentagem. Decimal
temperature Temperatura do ambiente em graus Celsius. Decimal
dew_point Temperatura do ponto de orvalho em graus Celsius. Decimal
heat_index Índice de calor em graus Celsius. Decimal
air_quality Nível de qualidade do ar em partes por milhão. Decimal
atmospheric_pressure Nível de pressão atmosférica em Pascal. Decimal
water_flow Nível de vazão de água em litros. Decimal
amount_water Nível de água do reservatório em litros. Decimal
percentage_water Nível de água do reservatório em porcentagem. Decimal
luminosity Nível de luminosidade do ambiente em lux. Decimal
ultraviolet_radiation Nível de radiação ultravioleta em miliwatts por centímetro quadrado. Decimal
created_at Data e hora da criação do registro em ISO 8601. Texto
updated_at Data e hora da última edição do registro em ISO 8601. Texto

Para enviar os dados dos sensores para a plataforma, os headers da requisição devem conter os seguintes dados:

Atributo Conteúdo Descrição Tipo
Content-Type application/x-www-form-urlencoded Definição do tipo de codificação dos dados. Texto
Authorization {{ api_token }} Token de autenticação da API. Texto

Para enviar os dados dos sensores para a plataforma, o payload da requisição deve conter os seguintes dados:

Atributo Descrição Tipo
rain_level Nível de chuva em porcentagem. Inteiro
soil_moisture Nível de umidade do solo em porcentagem. Inteiro
humidity Nível de umidade do ar em porcentagem. Decimal
temperature Temperatura do ambiente em graus Celsius. Decimal
dew_point Temperatura do ponto de orvalho em graus Celsius. Decimal
air_quality Nível de qualidade do ar em partes por milhão. Decimal
atmospheric_pressure Nível de pressão atmosférica em Pascal. Decimal
water_flow Nível de vazão de água em litros. Decimal
water_level Nível de água do reservatório em centímetros. Decimal
luminosity Nível de luminosidade do ambiente em lux. Decimal
ultraviolet_radiation Nível de radiação ultravioleta em índice ultravioleta. Decimal

Ao realizar uma requisição com o método POST, você poderá obter as seguintes respostas:

  • 201
  • 400
  • 401

Se os dados foram enviados com sucesso:

{
    "status": 201,
    "message": "Data has been sent successfully."
}

Se algum dado estiver com o formato inválido:

{
    "status": 400,
    "message": "Some sensor data is in an invalid format."
}

Se algum dado estiver incompleto:

{
    "status": 400,
    "message": "Sensor data is incomplete."
}

Se algum dado conter valores infinitos:

{
    "status": 400,
    "message": "Sensor data cannot have infinite values."
}

Se ocorreu algum erro ao salvar os dados:

{
    "status": 400,
    "message": "An error occurred while saving the data."
}

Se o token da API não foi informado:

{
    "status": 401,
    "message": "Enter the API token."
}

Se o token da API for inválido:

{
    "status": 401,
    "message": "Please enter a valid API token."
}
Atributo Descrição Tipo
status Código de estado da requisição HTTP. Inteiro
message Mensagem de resposta da requisição HTTP. Texto

Agendas de Irrigação
/api/irrigation-schedules/
id

O parâmetro id é opcional e deve ser usado para obter uma agenda de irrigação específica.

Atributo Descrição Tipo
id ID da agenda de irrigação. Inteiro

Para obter os dados das agendas de irrigação, você poderá utilizar este endpoint e o método HTTP abaixo.

  • GET

Ao realizar uma requisição com o método GET, você poderá obter as seguintes respostas:

  • 200
  • 404

Se houver registros no banco de dados:

[
    {
        "model": "core.scheduleirrigation",
        "pk": 1,
        "fields": {
            "irrigation_type": "time",
            "irrigation_period": "unique",
            "status": "irrigated",
            "datetime_on": null,
            "datetime_off": null,
            "time_on": "06:07:00",
            "time_off": "06:07:20",
            "humidity_on": null,
            "humidity_off": null,
            "night_irrigation": null,
            "water_flow": null,
            "created_at": "2023-01-11T06:06:43.502",
            "updated_at": "2023-01-11T06:07:24.265"
        }
    }
]
Atributo Descrição Tipo
model Nome da classe no banco de dados. Texto
pk Identificador do registro. Inteiro
fields Campos da classe no banco de dados. Objeto
irrigation_type Tipo de irrigação. Texto
irrigation_period Período de irrigação. Texto
status Estado da agenda de irrigação. Texto
datetime_on Data e hora que o sistema de irrigação deve ser ligado em ISO 8601. Texto
datetime_off Data e hora que o sistema de irrigação deve ser desligado em ISO 8601. Texto
time_on Hora que o sistema de irrigação deve ser ligado em ISO 8601. Texto
time_off Hora que o sistema de irrigação deve ser desligado em ISO 8601. Texto
humidity_on Umidade do solo que o sistema de irrigação deve ser ligado em porcentagem. Inteiro
humidity_off Umidade do solo que o sistema de irrigação deve ser desligado em porcentagem. Inteiro
night_irrigation Indica se o sistema de irrigação deve ser ligado a noite. Booleano
water_flow Vazão de água que o sistema de irrigação deve irrigar em litros. Decimal
created_at Data e hora da criação do registro em ISO 8601. Texto
updated_at Data e hora da última edição do registro em ISO 8601. Texto

O atributo irrigation_type pode possuir os seguintes valores:

Valor Descrição
day Por dia
time Por horário
humidity Por umidade
flow Por vazão

O atributo irrigation_period pode possuir os seguintes valores:

Valor Descrição
unique Único
daily Diário

O atributo status pode possuir os seguintes valores:

Valor Descrição
irrigated Irrigado
irrigating Irrigando
scheduled Agendado

Se a agenda de irrigação não existir:

{
    "status": 404,
    "message": "This irrigation schedule does not exist."
}
Atributo Descrição Tipo
status Código de estado da requisição HTTP. Inteiro
message Mensagem de resposta da requisição HTTP. Texto
